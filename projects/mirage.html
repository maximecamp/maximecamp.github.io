Mirage

Mirage est une application mobile et une extension Chrome développée dans le but de contourner les restrictions d'accès à des contenus, en fonction des règles définies par l'utilisateur.

MobileReact NativeWebJavascriptExtension Chrome

Une solution multi-plateformes pour contourner les restrictions d'accès

L'objectif de Mirage était de développer un outil permettant aux utilisateurs de naviguer sur le web selon leurs propres règles, en contournant les restrictions d'accès au contenu. Pour couvrir les cas d'usage mobiles et de bureau, j'ai développé ce projet sur deux plateformes distinctes. Une application mobile (React Native) et une extension Chrome (JavaScript). Ce projet démontre ma capacité à concevoir des outils complexes fonctionnant sur différentes plateformes, en utilisant des technologies adaptées à chaque environnement.

Application mobile (iOS & Android)

  • J'ai utilisé React Native avec TypeScript pour le développement cross-platform, permettant de partager une base de code majeure entre iOS et Android.
  • Pour les fonctionnalités spécifiques de navigation et filtrage, j'ai intégré des modules natifs personnalisés en Kotlin (Android) et Swift (iOS) afin de piloter finement la webview et les requêtes HTTP.

Extension Chrome (Navigateur de bureau)

  • Pour une intégration légère et très performante sur le navigateur, j'ai développé l'extension en JavaScript pur.
  • L'utilisation de l'API Chrome a permis une mise en œuvre efficace et directe des fonctionnalités de filtrage, adaptées à la simplicité requise par ce cas d'usage.

Ce projet met en avant mon expertise en développement React Native, ma capacité à intégrer du code natif (Kotlin/Swift) pour des besoins spécifiques complexes, ainsi que ma maîtrise du JavaScript et des API de navigateur.
